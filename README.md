# Quiz
A quiz containing a ranking system, a module for adding questions and an admin panel.

## Technologies 
- Android Studio
- Java
- Retrofit

## Features
- User registration and login process.
- The navigation is based on drawer navigation.
- Protection against no Internet connection.
- A dynamic quiz panel with a wide range of questions.
- Ability to add questions to the database.
- Ranking and statistics panel.
- Simple management of questions added by users in the admin panel.

## Setup
To run the applications you must:
1. Clone the server repository: [https://github.com/SzymonT99/PPSM_Quiz_Server.git](https://github.com/SzymonT99/PPSM_Quiz_Server.git)
2. Launch XAMPP Control Panel.
3. Import the MySQL database located on the server.
4. Run Spring Boot on the server.
